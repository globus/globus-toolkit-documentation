Globus Toolkit 3.2.1 Download

Globus Toolkit 3.2.1 is an incremental release of the Globus Toolkit containing updates to the 3.2.0 release.

GT3.2 implements services with a combination of C and Java. The C components run on Unix platforms, including Linux.

  • All Services (installs all web services and pre-web services components, including Core)
  • WS Base Installer (web services components, including Core)
  • Pre-WS Installer (pre-web services components, installer)
  • Pre-WS Bundles (pre-web services components, GPT bundles)
  • WS Core (pure Java)
  • WS Gars (pure Java)
