Activity - deliverToGFTP

Activity Type: Delivery

Description

Deliver data to a GridFTP server.

Example

<deliverToGFTP name="delivery">
  <fromLocal from="dataSource"/>
  <toGFTP host="somehost.org" port="8080" file="path/file.ext"/>
</deliverToGFTP>

Specification

Element deliverToGFTP:

  • Attribute name - unique name for the activity.
  • fromLocal (required) - input stream.
    • Attribute from - name of an input stream providing the data to deliver.
  • Element toGFTP (required) - the GridFTP server to which the data is to be delivered.
    • Attribute host (required) - GridFTP server host.
    • Attribute port (required) - GridFTP server port.
    • Attribute file (required) - GridFTP server path and (optional) file.
