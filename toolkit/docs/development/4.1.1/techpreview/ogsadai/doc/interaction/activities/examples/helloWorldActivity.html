Activity - helloWorldActivity

Activity Type: Example

Description

A simple activity that outputs "Hello World".

Example

<helloWorldActivity name="myActivity">
  <output name="helloWorldOutput"/>
</helloWorldActivity>

Specification

Element helloWorldActivity:

  • Attribute name - unique name for the activity.
  • output (required) - output stream.
    • Attribute name - name of the activity's output stream.
