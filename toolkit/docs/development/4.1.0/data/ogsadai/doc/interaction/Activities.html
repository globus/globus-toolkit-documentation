Activities

Activities are the data resource manipulation, data transformation and delivery actions that a client wants an OGSA-DAI service to perform. Some activities are data resource-specific (e.g. relational or XML activities), others (e.g. delivery and data transformation) are generic.

Activities are the basic building block of Perform documents.

Activities are designed to inter-operate. For example, the output of an sqlQueryStatement can be directed to a deliverToURL activity thereby allowing data to be delivered to third parties. To support inter-operation an activity can have zero or more inputs and zero or more outputs. These outputs can be given specific names and are termed streams.

Inter-operating activities

OGSA-DAI Activities

OGSA-DAI provides the following activities:

Activity Description
Relational Activities
sqlQueryStatement Run an SQL query statement.
sqlUpdateStatement Run an SQL update statement.
sqlStoredProcedure Invoke an SQL stored procedure. Currently, this only works with DB2 and is not supported by MySQL or Oracle.
sqlBulkLoadRowSet Bulk load data into a table.
relationalResourceManagement Create or drop database tables.
XML Activities
xPathStatement Run an XPath statement against an XML database.
xUpdateStatement Run an XUpdate statement against an XML database.
xmlCollectionManagement Create or remove collections within an XML database.
xmlResourceManagement Create or remove resources within an XML database.
Delivery Activities
deliverFromURL Retrieve data from a URL.
deliverToURL Deliver data to a URL.
deliverToGDT Push data to a service implementing the GridDataTransport portType.
deliverFromGDT Pull data from a service implementing the GridDataTransport portType.
deliverToGFTP Deliver data to a GridFTP server.
deliverFromGFTP Retrieve data from a GridFTP server.
deliverFromFile Retrieve data from a file on the container's local file system.
deliverToFile Deliver data to a file on the container's local file system.
deliverToSMTP Deliver data in an e-mail using SMTP.
inputStream Receive data through this service's GridDataTransport portType.
outputStream Deliver data through this service's GridDataTransport portType.
deliverToStream Deliver results to a stream
Transformation Activities
zipArchive ZIP compress the results.
gzipCompression GNU-ZIP compress the results.
gzipDecompression GNU-ZIP decompress results.
dataStore Cache parameters and results.
xslTransform Transform data using an XSLT.
stringTokenizer Break a single block into multiple blocks based on a set of separator characters. (Unsupported)
blockAggregator Aggregate multiple blocks into a single block. (Unsupported)
BinX Activities
fileQuery Query a BinX file.
File Activities
directoryAccessActivity Access a directory. (Unsupported)
fileAccessActivity Read data from a file. (Unsupported)
fileManipulationActivity Manipulate files in a directory. (Unsupported)
fileWritingActivity Write data into a file. (Unsupported)
Indexed File Activities
addIndexFile Import a new text file and create an index on it. (Unsupported)
searchIndexedFile Perform a full-text search across an indexed file. (Unsupported)
readFile Read data from a file. (Unsupported)
Miscellaneous Activities
notification Send or receive notifications as to the execution of other activities. (Unsupported)
Example Activities
helloWorldActivity A simple test example.

Note that activities marked Unsupported are activities that should work but have not been subjected to rigorous testing.
