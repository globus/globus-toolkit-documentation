OGSA-DAI WS-RF User Guide (v0.9.5)

Welcome to the OGSA-DAI user guide. OGSA-DAI is a middleware product which supports the exposure of data resources, such as relational or XML databases, on to Grids. Various interfaces are provided and many popular database management systems are supported. The software can also be extended to provide new functionality. For the "hands-on" user, feel free to jump straight into the Installation Guide.
