Activity - deliverToURL

Activity Type: Delivery

Description

Deliver data to a URL.

Example

<deliverToURL name="delivery">
  <fromLocal from="dataSource"/>
  <toURL>ftp://some.destination.org/file/file.ext</toURL>
</deliverToURL >

Specification

Element deliverToURL:

  • Attribute name - unique name for the activity.
  • fromLocal (required) - input stream.
    • Attribute from - name of an input stream providing the data to deliver.
  • Element toURL (required) - the URL to which the data is to be delivered. This URL should be of form:
    • scheme://[username:password@]host[:port]/location[?parameters] .
    • scheme can be one of http, https, ftp.
